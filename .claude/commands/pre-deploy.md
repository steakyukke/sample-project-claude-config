skill: deployment を参照して、デプロイ前チェックを実行してください。

手順:
1. コード品質チェック
   - `npm run lint` を実行
   - `npm run build` を実行
   - `npm run test` を実行

2. セキュリティチェック
   - `npm audit` を実行
   - ハードコードされた機密情報がないか確認

3. 環境変数チェック
   - 必要な環境変数が定義されているか確認
   - .env.example と比較

4. データベースチェック
   - マイグレーションファイルの確認
   - RLS 設定の確認

5. 結果レポート
   - すべてのチェック結果をまとめて報告
   - 問題があれば修正方法を提示

出力形式:
```
## デプロイ前チェック結果

### コード品質
- [x] Lint: 成功
- [x] Build: 成功
- [x] Test: 成功

### セキュリティ
- [x] npm audit: 脆弱性なし
- [x] 機密情報: ハードコードなし

### 環境変数
- [x] NEXT_PUBLIC_SUPABASE_URL
- [x] NEXT_PUBLIC_SUPABASE_ANON_KEY
- ...

### 結論
デプロイ可能 / 要修正
```
